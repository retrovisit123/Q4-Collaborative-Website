>git add . 
- Adds all changes to staging area

>git commit -m "commit"
- Saves all changes in staging area to the document

>git push origin
- Pushes saved changes to the main repository

>git pull
- Loads changes pushed by other collaborators documents

>git branch
- Check all the branchs you have

>git checkout -b <branch name>
- Create a branch and automatically get into it

>git checkout <branch name>
- Move to the choosen branch

>git branch -d <branch name>
- Safe delete a branch

>git branch -D <branch name>
- Force delete a branch (Not recomended unless really necessary)

>git merge <branch name>
- Merge all the changes into the choosen branch

>git pull origin <branch name>
- Pulls all the changes from specified branch as long as its in the main repository

>git branch "branchName"
- Creates a branch with the name specified in quotation marks

>git checkout "branchName"
- Switches the branch that you are working on to a different branch with the name specified in quotation marks
